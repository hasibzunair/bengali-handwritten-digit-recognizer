## What is it about?

Sixth place solution to kaggle bengali handwritten digit recognition. Achieved 97.5%. A VGG16 architecture with fine tuning and data augmentation. 

## Data Set  
* [Numta DB](https://www.kaggle.com/c/numta/data)
